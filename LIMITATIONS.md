# Grafana Cookbook Limitations

## Supported Platforms

Grafana is officially supported on the following operating systems:

* **Linux:**
    * **Debian / Ubuntu:** Debian 9+, Ubuntu 18.04 LTS+
    * **RHEL / Fedora / CentOS:** RHEL 8+, Fedora 34+, CentOS 8+
    * **SUSE / openSUSE**
    * **Alpine Linux:** 3.23+
* **Windows:** Windows 10 or later, and Windows Server 2016 or later.
* **macOS:** macOS 10.13 or later.
* **FreeBSD:** Supported within the FreeBSD lifecycle.

## Supported Architectures

Grafana provides binaries and packages for the following CPU architectures:

* **AMD64 / x86_64:** Supported on Linux, Windows, macOS (Intel), and FreeBSD.
* **ARM64 / AArch64:** Supported on Linux and macOS (Apple Silicon).
* **ARMv7 / ARMv6:** Often available for Linux (e.g., Raspberry Pi).

## Hardware Requirements (Minimum)

* **CPU:** 1 core (Minimum), 2+ cores (Recommended for production).
* **Memory:** 512 MB (Minimum), 2-4 GB (Recommended for production).
* **Disk Space:** 1 GB (Minimum), 10-20 GB SSD (Recommended for production).
